== Description ==

trend.ly is a website that helps people to follow, discover the trends all over the world in realtime and easy. this plugin enables you to embed any poll from trend.ly into your wordpress website. you can select unlimited number of polls and the plugin shows them randomly.

== Installation ==

1. Extract zip file to `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `trend.ly` in your templates
